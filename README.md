# Atta Studio

Atta Studio is a modern, AI-powered code editor built for hardware and software engineers. It features native support for KiCad schematics, PCB layouts, Gerber files, STEP/DXF models, Office documents, and more — all within a familiar VSCode-like interface.

## Features

- **AI Agent Integration** — Built-in chat panel powered by AttaOS for intelligent code assistance
- **KiCad Suite** — Schematic viewer/editor, PCB 2D/3D viewer, Gerber/Excellon overlay, BOM generation, DRC, ERC
- **CAD/CAM Support** — STEP and DXF file viewers with 3D rendering
- **Office Documents** — DOCX, XLSX, PPTX viewer (read-only rendering)
- **Mind Map Editor** — Create and edit mind maps with Markdown/XMind import/export
- **PRD & Spec Editor** — Structured product requirement and specification documents
- **Full IDE Features** — Monaco Editor, LSP, integrated terminal, Git panel, plugin system
