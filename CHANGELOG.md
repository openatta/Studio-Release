# Changelog

All notable changes to Atta Studio will be documented in this file.

## [Unreleased]

### Added
- VSCode-like editor with Monaco Editor and LSP support
- KiCad schematic viewer/editor with ERC, annotation, netlist export
- KiCad PCB 2D/3D viewer with DRC, interactive router, BOM generation
- Gerber and Excellon file viewers with multi-layer overlay
- STEP and DXF 3D model viewers
- Office document viewer (DOCX, XLSX, PPTX)
- Mind map editor with Markdown and XMind import/export
- PRD and Spec structured document editors
- AI chat panel integrated with AttaOS
- Plugin system with frontend-native React components
- Integrated terminal, Git panel, file explorer
- Multi-platform builds: macOS (arm64/x64), Windows, Linux
